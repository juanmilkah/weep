# Weep

A commandline password manager 

## Features 
- Add a Password
- Retrieve a Password 
- Update a Password 
- Delete a Password 
- Change Master Key
- List Passwords

# Installation

## Compiling from source
- Clone repository
- Install [Rust](https://www.rust-lang.org/)
- Install Cross 

```bash
cargo install cross
```

- Build the project
```bash
bash build.sh
```
